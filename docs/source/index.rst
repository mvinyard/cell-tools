Documentation
********************
``cell-tools`` is a package of helper functions for single-cell omics data analysis. ``cell-tools`` does not offer any novel methodology for analyzing single-cell data, rather it aims to collect useful pre- and post-processing functions and centralize them to enable swift single-cell data analysis. This packages is used and maintained by `Michael E. Vinyard<https://michaelvinyard.com/>` at Harvard University (research with Gaddy Getz and Luca Pinello at the Broad Institute and MGH). 

There is no paper to cite, however if you find this package useful for your own research, please consider reaching out and connecting.
